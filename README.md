# 天国拯救2炼金配方检索

一个简洁美观的天国拯救2（Kingdom Come: Deliverance II）炼金配方查询工具，帮助玩家快速查找和检索各种药剂、毒药和火药的制作配方。

## 功能特点

- **药剂名称查询**：输入药剂名称快速定位配方
- **材料查询**：通过材料名称搜索包含该材料的配方
- **多材料筛选**：选择多项材料，查找包含任意选中材料的配方
- **底料筛选**：按底料类型（水、烈酒、葡萄酒、油）单选过滤
- **详细配方展示**：点击卡片查看完整的炼金步骤和效果说明
- **响应式设计**：支持桌面和移动设备访问

## 包含配方

涵盖游戏中全部27种配方：
- **药剂**：解酒药剂、亨家私酿酒、伊索药剂、公鸡药剂、冥河之水、薄荷香水、夜鹰药剂、射手药剂、巧手药剂、救世干酒、止痛药剂、活络擦剂、消化药剂、狐狸药剂、甘菊药剂、生命之水、肥皂、艾蒿药剂、退烧药剂、金盏花药剂、雄狮香水、雄鹿之血药剂
- **毒药**：安眠毒药、玩偶毒药、祸根毒药
- **其他**：散弹火药、铅弹火药

## 使用方法

1. 直接在浏览器中打开 https://kiritoxf.github.io/kingdom-come-deliverance-2-alchemy-guide/
2. 使用搜索框输入关键词进行查询
3. 点击材料标签或底料标签进行筛选
4. 点击配方卡片查看详细炼金步骤
